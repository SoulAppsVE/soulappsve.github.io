# Pink Petal Video Page

This project is a visually appealing web page featuring falling pink petals and a video player at the center. The design utilizes a soft color palette to create a serene and engaging user experience.

## Project Structure

```
pink-petal-video-page
├── index.html          # Main HTML document
├── src
│   ├── styles
│   │   └── main.css    # CSS styles for the web page
│   ├── scripts
│   │   ├── main.js      # Main JavaScript file for initialization
│   │   └── petals.js    # JavaScript for falling petals effect
│   └── components
│       └── video-player.js # Video player component
├── videos
│   └── sample-video.mp4 # Sample video for the player
├── package.json         # npm configuration file
├── .gitignore           # Files and directories to ignore by Git
└── README.md            # Project documentation
```

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd pink-petal-video-page
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Open `index.html` in your web browser to view the page.
2. The video player will be displayed in the center, and pink petals will fall from the top of the page, creating a beautiful visual effect.
